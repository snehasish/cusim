Detailed Description

Basic data structure used in the LRU Queue.

The cacheBlock maintains the information we need regarding a specific block in the LRU Queue. The blocks are flexible in size. All addresses are word aligned.

uint64_t cacheBlock::endAddress

The word aligned end address of the cacheBlock.

The end address of the cacheBlock is inclusive. Eg: For a block starting at address 0x0 with size 8 words, let WORD_SIZE is 8 then end address is = 0x0 + 0x38 = 0x38

uint32_t* cacheBlock::utilizationBitmap

Integer pointer to first element of utilizationBitmap array.

The array stores information about which words in the block have been touched. Each element in the array stores the number of times that specific word has been accessed.
